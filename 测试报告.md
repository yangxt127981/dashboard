## 修复建议

### BUG-001：后端缺少 functionName 必填校验

**修复位置**：`RequirementServiceImpl.java` → `create()` 方法顶部添加：

```java
if (requirement.getFunctionName() == null || requirement.getFunctionName().isBlank()) {
    throw new IllegalArgumentException("需求名称不能为空");
}
```

同时在 `RequirementController.create()` 捕获并返回 400：

```java
} catch (IllegalArgumentException e) {
    return Result.error(400, e.getMessage());
}
```
